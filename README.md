# Holiday Spend Prediction

This project uses carefully simulated data to explore and model holiday spending behavior.  
It combines Python-based machine learning with R (Quarto) analysis and presentation to demonstrate an end-to-end data science workflow: data generation, preprocessing, modeling, interpretation, and communication.

## Project Overview

The goal of this project is to predict customer holiday spending and identify the most influential features driving purchase behavior. The project emphasizes not only model performance, but also interpretability and clear communication of results.

Key components include:
- Simulated customer transaction data inspired by a retail setting  
- Feature engineering and preprocessing  
- XGBoost regression modeling  
- SHAP-based feature importance analysis  
- Quarto-written technical analysis and presentation slides  

## Tools & Technologies

- Python (pandas, scikit-learn, XGBoost, SHAP)  
- R & Quarto  
- Jupyter Notebook  
- HTML slides (Quarto / revealjs)  

## 📁 Repository Contents

- `costco_basket.ipynb` – Python notebook for data processing, modeling, and evaluation  
- `XGboost.py` – standalone modeling script  
- `Costco_Holiday_Spend.qmd` – Quarto analysis document  
- `Costco_Slides.qmd` – Quarto slide source  
- `Costco_Slides.html` – rendered presentation slides  
- `Costco_simulated_data.csv` – simulated dataset  
- `Costco_SHAP.png` – SHAP feature importance visualization  

##  Key Skills Demonstrated

- Translating a business-style question into a predictive modeling problem  
- Building and tuning a machine learning regression model  
- Applying interpretable ML techniques (SHAP)  
- Bridging Python modeling with R-based reporting  
- Communicating technical findings through professional slides
